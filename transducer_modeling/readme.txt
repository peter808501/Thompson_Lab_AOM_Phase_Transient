This folder contains the modeling of the transducer network. 
The "field_fox_data" folder contains experimental data collected from a Thorlabs Field Fox Vector Network Analyzer. 
The "spice_optimization" folder contains a jupyter notebook that optimizes circuit parameters using spice, as well as the necessary spice files. 
The "transient_output" folder contains the transient phase and amplitude output of the final optimized spice file.

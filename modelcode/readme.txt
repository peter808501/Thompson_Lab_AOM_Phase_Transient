This folder contains 
1. the model used to simulate phase versus time and amplitude versus time graphs (model.ipynb)
2. the tutorial to the model (modeltutorial.ipynb)
3. a notebook containing graphs comparing simulated data to actual experimental data (graphs.ipynb)
4. simulated results produced by model.ipynb using experimental data in the folder 7.10Data (Simulated Results folder)
5. offcenter_frequency folder: experimental raw data for frequency \pm 5 MHz of 110 MHz.

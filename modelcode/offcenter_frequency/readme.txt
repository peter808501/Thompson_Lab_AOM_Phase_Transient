This folder contains the raw experimental data (voltage versus time) for the hetorodyne signal at off-center frequencies. 

Naming convention: the name if the frequency of the RF drive of the AOM. I start at 110MHz (center-frequency), then collect data down until 105, then 
back up to 115, and finally jump to 110. So there are 3 data files for 110MHz, one for the original, 2 for flybacks.
